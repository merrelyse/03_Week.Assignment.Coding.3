JavaScript Class Creation: MIDIengine

In this assignment, you will develop a JavaScript class named MIDIengine. This class will represent a fictional MIDI engine with input and output functionalities. Your task is to define the class in the script.js file and ensure it meets the specified requirements.

Accessing the Assignment Materials

  1. Navigate to the assignment repository in the course GitHub organization.
  2. Click on the green Use this template button in the top right corner.
  3. Choose 'Create a new repository'.
  4. Set up the repo name under your account.
  5. Click Create repository.

You should now be at your own personal repository with the assignment materials.

  1. Click on the green <> Code button and select Open with GitHub Desktop.
  2. Make sure the local path is where you would like to save this repo.
  3. Hit Clone.

  5. After working on your project, commit changes and push them to GitHub.

Objectives

Define a new class prototype called MIDIengine with the following features:

Constructor

  • Takes two parameters: input and output.
  • Initializes properties:
    • MIDIin (set to input)
    • MIDIout (set to output)
    • running (initially set to false)

Methods

  • enable(on_off): Turns the MIDI engine on or off and logs a message.
  • printInfo(): Logs the input and output details.

Testing (Using VS Code and your Browser)

Run the following tests in the JavaScript console to verify functionality:

  • testMIDI instanceof MIDIengine: Should return true.
  • testMIDI2 instanceof MIDIengine: Should return true.
  • MIDIengine.length: Should return '2'.
  • testMIDI.hasOwnProperty("MIDIin"): Should return true.
  • testMIDI.hasOwnProperty("MIDIout"): Should return true.
  • testMIDI.hasOwnProperty("running"): Should return true.
  • typeof testMIDI.enable: Should return function.
  • testMIDI.enable(true): Should log The MIDI Engine is now on.
  • testMIDI !== testMIDI2: Should return true.
